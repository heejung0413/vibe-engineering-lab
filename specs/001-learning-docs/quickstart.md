# Quickstart: 바이브코딩 및 개발 학습 문서

**Feature**: 001-learning-docs
**Date**: 2026-02-07

## 빠른 시작

### 1. 새 학습 문서 작성하기

1. 적절한 폴더로 이동
   - 바이브코딩 관련: `Vibe-Coding/[주제]/`
   - 개발 학습 관련: `개발-학습/[주제]/`

2. 새 마크다운 파일 생성
   ```bash
   touch Vibe-Coding/speckit/speckit-소개.md
   ```

3. 문서 템플릿 사용
   ```markdown
   # [제목]

   > 난이도: [입문/중급/고급]
   > 작성일: YYYY-MM-DD

   ## 학습 목표
   - 목표 1
   - 목표 2

   ## 개념 설명
   [내용]

   ## 실습
   [코드 예제]

   ## 정리
   - 요약 1
   ```

### 2. 기존 문서 수정하기

1. 수정일 업데이트
2. 변경 내용 반영
3. 코드 예제 동작 확인

### 3. 문서 간 링크 추가

상대 경로 사용:
```markdown
관련 내용은 [Claude Code 사용법](../claude-code/기본-사용법.md)을 참조하세요.
```

## 검증 체크리스트

- [ ] 한국어로 작성되었는가?
- [ ] 마크다운 형식을 따르는가?
- [ ] 코드 블록에 언어가 지정되었는가?
- [ ] 실습 예제가 포함되었는가?
- [ ] 학습 목표 → 개념 → 실습 → 정리 구조인가?
